Hexo+github静态托管搭建个人博客

参考文章:小茗同学

由于文章已经非常完善，笔者只提几点注意事项（均为笔者踩的坑）

  • 关于_config.yml的配置，master要更换为main
  • 关于 url 配置，文章中没有提到，但是笔者在搭建时发现，url 对大小写要求很严格，如果你的 github 域名没有完全正确的填写，那么网站的样式是加载不出来的
  • CNAME 位置并不会影响构建，而且不知道为什么，笔者的 CNAME 没有放到source文件夹中，但并没有被转为 html 文件
